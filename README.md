# StockApp
Stock Price Web Application - Project Overview - This project is a simple stock price web application built with Python and Streamlit. The user enters a stock ticker symbol, selects a time period, and views stock data with a chart of closing prices.

## Tools Used
- Python
- Streamlit
- yfinance

## Files Included
- `app.py`
- `requirements.txt`
- `README.md`

## How to Install
```bash
pip install -r requirements.txt


## How to Run
conda activate streamlit
pip install -r requirements.txt
streamlit run app.py
